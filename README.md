# Antigravity Dashboard

Central hub for managing AI plugins, skills, tasks, git operations, and system controls.

## Features

- Plugin Manager - Enable/disable AI plugins
- Skills & Agents - Browse and manage AI skills
- Task Scheduler - Create and manage background tasks
- Git Integration - Commit and push from the dashboard
- Real-time Logs - SSE-based log streaming
- Session Analytics - Usage statistics and charts
- File Watcher - Monitor directory changes
- Backup & Restore - Zip/restore configuration
- Global Rules - Inject rules into AI context
- Subagents Builder - Create custom AI agents
- Online Store - Install community skills
- Custom Macros - Build slash command macros
- Memory Injector - Persistent project memory
- System Controls - Volume, media, file organizer, web scraper
- Agent Arena - Multi-agent AI debate system

## Setup

```bash
npm install
node plugin_manager.js
```

Visit `http://localhost:4000`
